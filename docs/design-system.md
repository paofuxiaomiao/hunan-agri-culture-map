# 湖南省农耕文化地图 - 设计系统文档

## 项目概述

湖南省农耕文化地图是一个以地图为核心载体的文化展示平台，采用"新中式文博展示设计"风格，融合传统书画卷轴美学与现代数字交互。整体以"古籍地图"为核心隐喻，通过宣纸纹理、金石色调、印章标识等视觉元素，传递湖湘农耕文明的厚重与温暖。

---

## 设计运动与核心原则

本项目的设计运动定位为**新中式文博展示设计**，核心原则包括四个维度：纸韵温润（宣纸米白为底，面板带纸质纹理）、金石雅致（棕金色为主色调）、留白呼吸（大面积地图留白，信息面板轻盈浮动）、层次分明（半透明毛玻璃与柔和阴影建立信息层级）。

## 布局系统

页面采用经典的全屏单页布局，垂直方向分为四个区域：

1. **顶部导航区**（56px高）：品牌印章Logo + 导航Tab + 搜索/分享按钮
2. **统计横栏**（36px高）：四项核心数据指标展示
3. **地图主视区**（flex-1自适应）：左侧筛选浮层 | 中央Leaflet地图 | 右侧详情浮层
4. **底部模块区**（175px固定高）：四栏内容（主题线路/发展脉络/节气/文物）
5. **页脚**（~65px）：深绿色背景 + 备案信息

整体使用 `h-screen flex flex-col` 实现全屏不滚动布局。

## 组件材质语言

### 面板材质（glass-panel）
所有浮动面板统一使用"宣纸+金线"材质：
- 背景：`rgba(255, 253, 248, 0.94)` + `backdrop-filter: blur(16px)`
- 边框：`1px solid rgba(139, 105, 20, 0.12)`
- 阴影：`0 4px 24px rgba(44, 36, 24, 0.06)` + 内发光

### 金色分隔线（gold-thread）
- `linear-gradient(90deg, transparent 0%, rgba(139, 105, 20, 0.25) 20%, ... 80%, transparent 100%)`

### 地图标记样式
- 古代遗址：棕金色圆点 + 金色标签
- 现代地标：山水绿圆点 + 绿色标签
- 红色旧址：印章红圆点 + 红色标签
- 标签字体：Noto Serif SC，12px，600 weight

### 操作按钮（圆形金色）
- 尺寸：44x44px 圆形
- 背景：`linear-gradient(135deg, #a8872e 0%, #8B6914 100%)`
- 阴影：`0 2px 8px rgba(139,105,20,0.3)`
- 图标：白色 Lucide icons，18px

## 装饰元素

### 底部模块装饰
- 背景渐变：`linear-gradient(180deg, #faf8f2 0%, #f0e8d4 100%)`
- AI生成田园山水背景图（opacity 12%，mixBlendMode: multiply）
- 各headline前添加水稻图标装饰

### 页脚装饰
- 深绿色背景 `#2d4a3e`
- 几何花纹纹理叠加（opacity 8%）
- 山水剪影SVG图标

### 地图区域
- 宣纸纹理叠加层（radial-gradient模拟边缘晕染）
- 比例尺采用古籍地图风格
