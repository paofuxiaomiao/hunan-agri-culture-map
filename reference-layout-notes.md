# 参考图布局分析

## 整体结构（1672x941px 单屏布局）
整个页面在一屏内展示完毕，不需要滚动。

### 顶部区域 (约 90px)
- 第一行：深棕色渐变背景的导航栏，左侧红色印章logo+"湖南省农耕文化地图"标题+副标题，中间导航tabs（地图浏览/主题线路/发展脉络/重要文物/节气日历），右侧搜索框+分享按钮
- 第二行：浅色统计栏，居中显示4个统计项（文化点位128处/主题线路3条/重要文物24件/二十四节气24个）

### 中间地图区域 (约占60%高度，约560px)
- 地图占据中间主体区域
- 地图背景是暖色调的湖南省地图（有省界轮廓，非全屏地图瓦片）
- 左上角：地图控制按钮（+/-/定位/图层）
- 左侧浮动面板：图层筛选面板（约240px宽），包含图层控制（3个toggle开关）、点位搜索、图例说明、清空筛选按钮
- 右侧浮动面板：点位详情面板（约320px宽），包含标题、分类标签、封面图、描述文字、标签chips、底部操作按钮（查看详情/路线规划/一键分享）
- 右下角：小地图缩略图
- 左下角：比例尺

### 底部内容区域 (约占30%高度，约280px)
- 分为4栏，用竖线分隔
- 第1栏"主题线路"：3个横向卡片（图片+标题+描述），每个卡片是横向排列的
- 第2栏"湖湘农耕文化发展脉络"：纵向时间轴，每项有图标+标题+时间段
- 第3栏"二十四节气"：2行x4列的圆形图标网格，每个有植物图标+名称
- 第4栏"重要文物"：2行x3列的方形缩略图网格，每个有图片+名称

### 页脚 (约60px)
- 红色边框包围
- 主办单位信息
- ICP备案号和公安备案号

## 关键视觉特征
- 整体暖色调（米白/棕金/深棕）
- 地图有明显的省界轮廓
- 面板使用半透明白色背景
- 点位标签使用分类颜色（金色/绿色/红色）
- 底部模块紧凑排列，信息密度高
