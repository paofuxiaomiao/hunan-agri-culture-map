# 湖南省农耕文化地图 - 设计方案

## 参考图作为设计规范

本项目以用户提供的参考图为设计模板，在高度还原布局与功能的基础上，注重画面和谐、艺术美感与交互流畅性。

## 选定方案：湖湘古籍地图

### Design Movement
新中式文博展示设计 —— 融合传统书画卷轴美学与现代数字交互，以"古籍地图"为核心隐喻。

### Core Principles
1. **纸韵温润**：整体以宣纸米白为底，所有面板与卡片带有微妙纸质纹理
2. **金石雅致**：棕金色为主色调，传递农耕文明的厚重与温暖
3. **留白呼吸**：大面积地图留白，信息面板轻盈浮动，不压迫视觉
4. **层次分明**：通过半透明毛玻璃、柔和阴影与微妙边框建立清晰的信息层级

### Color Philosophy
- **主色**：棕金色 #8B6914 / oklch(0.52 0.12 75) —— 稻穗成熟的温暖
- **辅色**：山水绿 #4A7C59 —— 田野与山林的生机
- **点缀**：印章红 #C41E3A —— 文化印记的庄重
- **底色**：宣纸米白 #F5F0E8 —— 古籍纸张的温润
- **文字**：墨色 #2C2418 —— 毛笔墨迹的沉稳

### Layout Paradigm
参照参考图的经典三栏+底部模块布局：
- 顶部：导航条 + 统计卡片横栏
- 中部：左侧筛选浮层 | 中央地图主视区 | 右侧详情浮层
- 底部：四列内容模块横向排布
- 页脚：备案信息

### Signature Elements
1. **印章 Logo**：红色方印风格品牌标识
2. **宣纸纹理**：贯穿所有面板与背景的微妙纸质肌理
3. **金色描边**：卡片与面板边缘的细腻金色线条装饰

### Interaction Philosophy
- 地图交互：平滑缩放、拖拽浏览、点位悬停高亮、点击弹出详情
- 面板交互：左侧面板可收起/展开，右侧详情卡支持上下切换
- 图层开关：Toggle 切换带有流畅过渡动画
- 底部模块：卡片悬停微上浮，时间轴节点可交互

### Animation
- 面板展开/收起：300ms ease-out 滑动
- 点位标记：悬停时 scale(1.2) + 光晕脉冲
- 卡片入场：从底部 translateY(20px) + opacity 渐入
- 页面加载：地图区域先渲染，面板依次淡入
- 图层切换：点位以 200ms 交错动画显示/隐藏

### Typography System
- 标题字体：Noto Serif SC（思源宋体）—— 传统文化气质
- 正文字体：Noto Sans SC（思源黑体）—— 清晰可读
- 数字/统计：DIN Alternate / Roboto Mono —— 数据展示

### Brand Essence
一句话定位：湖南省农耕文化的数字展示窗口，面向文化爱好者与研学旅行者，以地图为媒介串联万年农耕文明。
性格形容词：**温润**、**厚重**、**雅致**

### Brand Voice
标题风格：简洁有力，带有文化底蕴感。
- 示例1："小小一幅地图，展开湖湘万年农耕文明"
- 示例2："稻作之源，文明之根"

### Wordmark & Logo
红色方形印章风格，内含"农耕文明"四字篆书变体，外框带有岁月侵蚀的不规则边缘。

### Signature Brand Color
**稻穗金** #8B6914 —— 成熟稻穗在阳光下的温暖金棕色，是整个品牌最具辨识度的颜色。

## Style Decisions

- 地图底图必须采用"宣纸古籍地图"视觉处理：低饱和、暖米白、墨线感道路与山水绿地貌，避免直接呈现现代通用地图的高信息噪声。
- 品牌区规则：红色方印与宋体标题共同构成主识别，页头不得呈现为普通后台导航，必须保留文博馆藏式的庄重感。
- 点位与内容图像规则：地图标记、图例与缩略图应统一采用"印章 / 稻穗 / 金石拓片"语汇，避免默认地图 pin、普通相册缩略图和泛用图标感。
- 所有面板、卡片统一使用"宣纸+金线"材质语言，避免随意使用现代圆角卡片。
- 字体系统：标题/节头/地图标注使用宋体（Noto Serif SC），数据和控件使用无衬线体（Noto Sans SC）。
